For the bank to work, you need to change Secret Pass.txt in the server folder: BankData to the password you want.
And also on each client in the Network library, replace the field: Network.ID with the ID of the computer on which the bank server is located

Чтобы банк работал вам нужно изменить Secret Pass.txt в папке сервера: BankData на тот пароль который вы хотите.
А так же на каждом клиенте в библиотеке Network заменить поле: Network.ID на ID компьютера на котором стоит сервер банка
